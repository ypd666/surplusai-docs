# 常见问题

## 应该选择哪个系统？

选择运行客户端的系统，而不是访问文档的设备系统。例如，在手机上查看文档、准备为 Windows 电脑安装客户端时，应选择 Windows。

## 为什么命令没有变化？

只有包含系统差异的命令块会随右上角选项变化。如果三个系统使用相同命令，切换后内容看起来不会改变。

## API 密钥丢失怎么办？

完整密钥通常无法再次查看。请创建新密钥，更新客户端配置，然后撤销旧密钥。

## 客户端返回鉴权失败怎么办？

确认密钥没有多余空格、没有被撤销，并检查客户端是否使用了正确的 SurplusToken 接入地址。

## 客户端无法连接怎么办？

依次检查账户余额、服务状态、接入地址、API 密钥和本地网络。仍无法解决时，保留错误信息和发生时间，以便进一步排查。
