# 介绍

[SurplusToken](https://surplustoken.com) 是基于 Sub2API 开发，面向北大校内师生的 Codex 模型调用服务。你可以在平台中管理账号、API 密钥、余额和拼车服务，并将 Codex、OpenCode 等客户端接入平台。

## 文档内容

- **平台功能**：注册账号、创建密钥、充值和使用拼车。
- **客户端接入**：配置 Codex CLI、Codex Desktop、Claude Code、Kimi Code 和 OpenCode。

## 使用前准备

开始前，请准备一个可正常接收邮件的邮箱。接入客户端时，还需要在平台中创建 API 密钥。

:::warning
API 密钥属于敏感凭证。请勿在聊天记录、截图、公开代码仓库或客户端日志中泄露密钥。
:::
