# 快速开始

按照以下步骤完成 SurplusToken 的基础配置。

<SystemSelector variant="content" />

## 1. 注册账号

打开 SurplusToken 平台，使用邮箱注册并登录账号。详细步骤参见[注册账号](/platform/register)。

## 2. 创建 API 密钥

进入密钥管理页面，创建一个用于客户端接入的 API 密钥。详细步骤参见[创建密钥](/platform/create-api-key)。

## 3. 确认余额

在计费页面确认账户余额。如余额不足，请先完成充值。详细说明参见[计费与充值](/platform/billing)。

## 4. 接入客户端

根据你使用的客户端继续配置：

- [Codex CLI & Desktop 接入](/clients/codex)
- [OpenCode 接入](/clients/opencode)

正文和页面右上角的系统选择器会保持同步。后续文档中的安装流程与命令会随系统选择自动变化。
