# 贡献账号

贡献账号用于将符合要求的账号接入 SurplusToken，供平台按照相应规则统一调度。贡献前，请仔细阅读页面展示的适用范围、收益规则和风险说明。

## 贡献前准备

- 确认账号来源合法，并且由你本人持有或已获得明确授权。
- 确认账号符合平台当前的贡献条件。
- 移除与贡献无关的敏感信息和第三方授权。
- 了解贡献期间的使用范围、结算方式和停止规则。

## 贡献步骤

1. 登录 SurplusToken 平台。
2. 进入贡献账号页面。
3. 选择需要贡献的账号类型。
4. 按照页面提示完成账号验证或授权。
5. 检查贡献规则并确认提交。
6. 等待平台完成状态检查。

贡献成功后，可以在贡献记录中查看账号状态、使用情况和收益明细。

:::warning
不要通过聊天、邮件或截图向他人发送账号密码、验证码、恢复代码等敏感凭证。仅在 SurplusToken 官方页面中完成授权。
:::

## 停止贡献

如需停止贡献，请在贡献记录中选择对应账号并解除授权。解除后，平台可能需要一定时间完成正在进行的任务和结算。

停止贡献后，建议前往账号提供方检查已授权的应用和活跃会话，并撤销不再需要的授权。

## 账号状态异常

账号出现验证失败、授权过期或不可用时，请根据页面提示重新验证。请勿短时间内反复提交，以免触发账号提供方的安全限制。
